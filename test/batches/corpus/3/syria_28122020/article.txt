هەرچەندە ' هەسەدە ' بەفەرمی هەواڵی رادەستکردنەکەی رانەگەیاندووە، بەڵام هێزەکانی سوپای تورکیا و میلیشیاکانی ' سوپای نیشتمانی ' لە بۆردوومانکردنی ناوچەکە بەردەوامن.  ئاژانسی ' سپۆتنیک ' ی رووسی لە زاری عومەر رەحمون، ئەندامی دەستەی ' پێکهاتنەوەی نیشتمانی ' سەر بە حکومەتی سووریای گواستەوە، "دوێنێ یەکشەممە کۆبوونەوەیەک کراوە و تێیدا رێکەوتنکرا لەسەر رادەستکردنی شاری عەین عیسا بەهێزەکانی حکومەتی سووریا و لە پاڵیدا لایەنی رووسی".  رەحمون رایگەیاندووە، "تائێستا وردەکاری تەواو لەبارەی بڕگەکانی رێکەوتنەکەوە بەردەست نییە، بەڵام کورد لایەنی رووسییان ئاگادارکردووەتەوە، کە ئەوان ئامادەن بۆ پاشەکشەکردن لە ' عەین عیسا ' بە تەواوی، هەنگاوە کردەنییەکانی سەر زەویش لە چەند کاژێری کەمی داهاتوودا دەستپێدەکات".  ژەنەراڵی دەریاوان ڤیاچیسلاف ستینیک، جێگری بەڕێوەبەری ' سەنتەری حمێمیم بۆ پێکهاتنەوەی لایەنە ناکۆکەکان لە سووریا ' کە سەر بە وەزارەتی بەرگری رووسیایە، ئاشکرایکردووە، "بەئامانجی پتەوکردنی هەوڵەکان بۆ بەرقەرارکردنی سەقامگیری لە ناوچەی ' عەین عیسا '، ئەمڕۆ یەکەی زیاتری پۆلیسی سەربازیی رووسیا، گەیشتنە ئەو ناوچەیە".  سیتنیک ئاماژەی بەوەداوە، دۆخی عەین عیسا ناسەقامگیرە، زیاتر دەڵێ: "پێشتر لە کاتی دانوستاندنەکاندا لەگەڵ لایەنی تورکیا، گەیشتینە رێککەوتن لەبارەی دانانی خاڵە پشکنینەکانی سووریا و رووسیا بەهاوبەشی"، بەڵام ئەو ئاماژەی بەڕێکەوتن نەداوە، کە ببێتە هۆکاری رادەستکردنی تەواوی شارەکە، بە رژێمی سووریا.  هێزەکانی سووریای دیموکرات، جیا لەوەی هێشتا هیچ کۆمێنتێکیان لەسەر ئەو رووداوە نەداوە، لەسەر زەویش هیچ جموجوڵێک بە ئاڕاستەی پاشەکشەکردنی هێزەکانی لە شارەکەدا هەستی پێنەکراوە.  سەرچاوەکانی هەسەدە کۆبوونەوەکەی دوێنێی نێوان خۆیان و رووس و سوورییەکانیان پشتڕاستکردووەتەوە، بەڵام ئاماژەیان بەوەداوە کە هیچ پێشهاتێکی نوێ لە کۆبوونەوەکەدا نەبووە، ئاماژەیان بەوەداوە، سوپای تورکیا و ' سوپای نیشتمانیی سووریا ' کە تورکیا پشتگیریدەکات، هێرشەکانیان بۆ سەر گوندی مەعلەک دەستپێکردووەتەوە، ئەوەشیان ئاشکراکردووە، بەرەبەیانی ئەمڕۆ دووشەممە، بۆردومانکردنی گوندەکەیان بەتۆپی قورس دەستپێکردووەتەوە.  سەرچاوەکان دەڵێن، تۆپخانەی تورکیا، هێزەکانی هەسەدەی لە نزیک گوندەکانی ' مشیرەفە ' و ' جەهبەل ' لە نزیک رێگەی نێودەوڵەتی رۆژهەڵاتی ' عەین عیسا ' بۆردومان کردووە، هەروەها تۆپبارانی ناوچەی ' سەوامیع ' و دەوروبەریان کردووە.  پێشتر تورکیا گوشاری زۆری لەسەر ' هەسەدە ' دروستکردووە، بۆ پاشەکشەکردن لە شاری ' عەین عیسا '، بەڵام ' هەسەدە ' لە سەرجەم کۆبوونەوەکانی لەگەڵ لایەنی رووسیدا، ئەو کارەی رەتکردووەتەوە، لەکاتێکدا تورکیا سوورە لەسەرئەوەی هیچ هێزێکی ' هەسەدە ' لە شارەکەدا نەمێنێتەوە.  لە چەند مانگی رابردوودا، بەرەکانی نزیک لە شارەکە هەڵچوونی بەردەوامیان بەخۆوە بینیووە، سوپای تورکیا و ' سوپای نیشتمانی سووریا ' هێرشدەکەن و ' هەسەدە ' ش بەرگرییەکانی خۆی لە بەرەکاندا پتەوتر کردووە و هێرشی پێچەوانەی بۆ سەر ئەو دوو سوپایە کردووە.  ' عەین عیسا '، پێگەیەکی ستراتیژی هەیە، لەنزیک رێگەی نێودەوڵەتیی حەلەب - حەسەکە، کە رۆژهەڵات و رۆژئاوای سووریا بەیەکەوە دەبەستێتەوە، دەشچێتەوە سەر دیمەشقی پایتەختی سووریا. شاری ' عەین عیسا ' بە پایتەختی کارگێڕیی هێزەکانی ' هەسەدە ' دادەنرێت لەوکاتەوە لە ساڵی ٢٠١٥ دا توانییان چەکدارانی داعش لە شارەکە دەرپەڕێنن.  پشێویی ناوچەکانی ژێر دەسەڵاتی ' هەسەدە ' بەردەوامە، دوێنێ شەو، بەرپرسێکی پێشووی ' هێزەکانی سووریای دیموکرات ' لە دەوروبەری دێرزۆر لەلایەن چەکدارانی نەناسراوەوە کوژرا، لە کاتێکدا لە هەمان ناوچە هێزەکانی ' هەسەدە ' هەڵمەتی دەستگیرکردنی چەندین ناڕازییان لە شارەکەدا ئەنجامدا.  ئەمڕۆش لە دوو تەقینەوەدا لە دەوروبەری حەلەب، خێزانێکی پێنج کەسی گیانیانلەدەستدا کە لە نێویاندا سێ منداڵ هەبوون، هاوکات تەقینەوەیەک لە سووتەمەنی فرۆشییەکدا لە دێرزۆر روویدا، هێشتا هۆکاری رووداوەکانیش ئاشکرانەکراون.
