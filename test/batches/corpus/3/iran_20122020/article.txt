ئێران بەئاشکرا دانی بەدروستکردنی دامەزراوە نوێیە ئەتۆمییەکەی لە ' فۆدرو ' نەناوە، بەڵام دەوڵەتانی رۆژئاوا لەسەر ئەو بۆچوونە سوورن، کە ئێران وێستگە نوێیەکەی دروستکردووە. ئاژانسی ' ئاسۆشێیتد پرێس ' وێنەی ئاسمانیی وێستگەکەی بڵاوکردەوە.  دەوڵەتانی رۆژئاوا، پێشتر ئاشکرایانکردبوو لە ساڵی ٢٠٠٩ دا، ئێران وێستگەکەی دروستکردووە، ئەوەیان لە میانی گەشتێکی پشکنیندا بۆ دەرکەوت لەسەر خاکی ئەو وڵاتە.  هەرچەندە ئامانجی دروستکردنی دامەزراوەکەی ' فۆدرو ' هێشتا روون نییە، بەڵام بڵاوبوونەوەی وێنە نوێیەکانی ئاژانسە جیهانییەکە، پێشبینی هەڵچوونی زیاتری لە نێوان ئەمریکا و ئێراندا هێناوەتە ئاراوە، بەتایبەت کە ئیدارەی دۆناڵد ترەمپ، دوا رۆژەکانی خۆی لە کۆشکی سپیدا بەڕێدەکات.  کۆمەڵگەی نێودەڵەوتی جیا لە پێگە نوێیەکە، ئاگاداری وردەکاریی کارەکانی ئێرانە، بۆ بونیادنانی دامەزراوەیەکی دیکەی ئەتۆمی لە ناوچەی نەتەنەز، دوای ئەوەی وێستگە ئەتۆمییەکەی پێشووی وڵاتەکە، بەهۆی تەقینەوەیەکی سەرچاوە نادیارەوە لە تەمووزی رابردوودا، چیدی کەڵکی بەکارهێنانی نەما.  ئێران ئەوکات تەقینەوەکەی نەتەنزی، بەهێرشێکی گێرەشێوێنانە، ناوبرد.  جیفری لویس، پسپۆری ' سەنتەری جەیمس مارتن بۆ توێژینەوەکانی بڵاوبوونەوەی ئەتۆمی '، لە پەیمانگای ' میدلبۆری بۆ توێژینەوە نێودەوڵەتییەکان '، دەڵێت "هەر گۆڕانکارییەک لەو پێگەیەدا بەوردی چاودێریدەکرێت، چونکە ئاماژەیە بۆ پڕۆگرامە ئەتۆمییەکەی ئێران". تائێستا ئێران لەرێگەی نوێنەرەکانییەوە لە نەتەوەیەکگرتووەکان، هیچ کۆمێنتێکی لەسەر وێنەکان نەداوە، ' ئاژانسی نێودەوڵەتی وزەی ئەتۆم ' یش، هێشتا قسەیەکی لەسەر وێنە بڵاوکراوەکان نەکردووە.  کاری دروستکردن لە پێگەی ' فۆدرو ' لە کۆتاییەکانی ئەیلوولدا دەستپێکردووە، وێنەی مانگە دەستکردەکان دەریدەخەن، کارکردن لە بەشی باکووری رۆژهەڵاتی پێگەکەدا لە نزیک شاری قومی شیعەنشین دەستیپێکردووە، کە ٩٠ کیلۆمەتر لە تارانەوە دوورە. وێنەگیراوەکەی مانگە دەستکردەکە لە ١١ی کانوونی یەکەم، دەریدەخات بناخەی هەڵکۆڵراوی پێگەکە لە باکووری رۆژهەڵاتی بەشەکانی ' فۆدرو ' دەستیانپێکردووە، دەیان دینگە دروستکراون، دەکرێ ئەو دینگانە بناخەی باڵەخانەکە بن بۆ پاراستنی لە بوومەلەرزە.  پێگەکە دەکەوێتە بەشی باکووری رۆژهەڵاتی ' فۆدرو '، لە ژێر زەویدا، لە قووڵایی چیادا دروستدەکرێت، ئەمەش بۆ پاراستنی لە گورزە ئاسمانییەکان. پێگەکە دەکەوێتە نزیک چەندین باڵەخانەی دیکەوە، بۆ پشتگیری و لێکۆڵینەوە و گەشەپێدان لە ' فۆدرو '، لەنێویاندا "سەنتەری نیشتمانیی ئێران بۆ تەکنەلۆژیای هەوا". تەکنەلۆژیای هەوا، پێکهاتەیەکی یەکلاکەرەوەیە لە ئامێرەکانی دەرکردەی ناوەندیی ئێراندا، کە گازی یۆرانیۆم بەکاردەهێنێت بۆ پیتاندن.  پێشتر پێگەیەکی سەر ' تویتەر ' بەناوی ' ئۆبزیرل ئیلینۆی ' وێنەی ئاسمانیی پێگەکەی بڵاوکردبووەوە، ئەو ماڵپەرە گووتبووی وێنەکە لە ' پەیمانگای توێژینەوەکانی بۆشایی ' کۆریای باشوور وەرگیراون. دواتر ئاژانسی ' ئاسۆشێیتد پرێس ' پەیوەندی بە سەرپەرشتیاری پێگەکەی توویتەرەوە کرد، ئەو خۆی بەوە ناساندووە کە سەربازێکی خانەنشینی هێزە بەرگرییەکانی سوپای ئیسرائیلە، باکگراوندێکی ئەندازەی مەدەنی هەیە. پاشان پەیمانگای توێژینەوەکانی بۆشایی کۆریا، دانی بە وێنەکاندا نا کە بڵاوکراونەتەوە.  ساڵی ٢٠١٨ ویلایەتە یەکگرتووەکانی ئەمریکا لە رێکەوتنی ئەتۆمی لەگەڵ ئێران کشایەوە. ئەوکات ترەمپ گوتی پڕۆگرامە ئەتۆمییەکەی ئێران و مووشەکە بالیستیەکانی ئەو وڵاتە و بابەتە ناوچەیی و نێودەوڵەتییەکانی دیکە، هۆکاری پاشەکشەی تاکلایەنەی ئەمریکان لە رێکەوتنەکە. ئێران لە دوای پاشەکشەی ئەمریکا و سەپاندنەوەی سزاکان بەسەریدا، رادەی پیتاندنی یۆرانیۆمی زیادکردووە، لە کاتێکدا دەبوو بەپێی رێکەوتنەکە بەڕێژەیەکی کەم ئەو کارە بکات.  پەرلەمانی ئێران، لە هەفتەکانی رابردوودا، لەپاش کوژرانی موحسین فەخریزادەی زانای ئەتۆمی لە نزیک تاران، داوای بەرەوسەربردنی پیتاندنی یۆرانیۆمی لە وڵاتەکەدا کرد، وەک پەرچەکردار بەرامبەر کوشتنی فەخریزادە.
