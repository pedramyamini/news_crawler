ئەلبوومەکە لە ١٢ تراکی گۆرانی پێکهاتووە و دوو لە گۆرانییەکان فۆلکلۆرن. ئاوازی گۆرانییەکان لەلایەن ئەیوب عەلییەوە دانراون، هەروەها تێکستی بۆ نۆ گۆرانیش نووسیوە، دوو تێکستی دیکەش لەلایەن شاعیر رەنج سەنگاوی نووسراون و تێکستی یەکێک لە گۆرانییەکانیش لە نووسینی جەبار حەبیبە.  گۆرانییەکان بەم ناوانەن، ' لێ لێ '، ' هیوا '، ' خەیاڵ '، ' بڵێنمان شکاند '، ' لێم مەپرسە '، ' کۆڵبەر '، ' بۆ دایکم '، ' نامەکانت '، ' رەنگ '، ' پەیکەر '، ' زاڵم ' و ' بەیانیت باش '.  ئەیوب عەلی لە ٢٤ ئازاری ئەمساڵ، یەکێک تراکەکانی ئەلبوومەکی بەناوی ' رەنگ ' بەشێوەی ڤیدیۆکلیپ بڵاوکردەوە و زیاتر لە ١١١ هەزار بینەری لە یوتیوب هەیە.
