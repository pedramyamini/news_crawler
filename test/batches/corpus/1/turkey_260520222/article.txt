ئەنجوومەنی سیاسەتی نەختینەیی دراوی بانکی ناوەندی تورکیا پاشنیوەڕۆی ئەمڕۆ پێنجشەممە ٢٦ی ئایاری ٢٠٢٢ پێنجەمین کۆبوونەوەی خۆی لەمساڵدا بە سەرۆکایەتی شەهاب کاڤجیئۆغڵو، سەرۆکی بانکی ناوەندی کرد.  بانکی ناوەندیی تورکیا بڕیاریدا بە دەستکارینەکردنی سوودی بانکی و بۆ پێنجەمینجار لەسەر یەک سوودی بانکی لە رێژەی ١٤ % دا بەجێگیری هێشتەوە.  دوای راگەیاندنی بڕیارەکە، بەهای لیرەی تورکی بەرامبەر بە دۆلار و دراوە بیانییەکان دابەزی. یەک دۆلاری ئەمریکی کە پێش بڕیارەکە بە ١٦.٣٧ لیرە مامەڵەی پێوەدەکرا بەهای لەدەستدا و هەر دۆلارێک لە بەرامبەر ١٦.٤٢ لیرەی تورکی مامەڵەی پێوەکرا.  سوودی بانکی لە تورکیا لە سەرەتای ساڵی ٢٠٢١ دا ١٧ % بوو. بانکی ناوەندیی تورکیا لە ١٨ی ئاداری ٢٠٢١ دا، سوودی بانکی لە ١٧ % ەوە بۆ ١٩ % بەرزکردەوە و دواتر لە پێنج کۆبوونەوەدا ئەو رێژەیەی جێگیر کرد.  لە ٢٣ی ئەیلولی ٢٠٢١ رێژەی سوودی بانکی لە ١٩ % بۆ ١٨ % کەمکرایەوە. لە ٢١ی تشرینی یەکەمی ٢٠٢١، رێژەی سوودی بانکی بۆ ١٦ % و لە ١٨ی تشرینی دووەمیشدا رێژەکەی لە ١٦ % ەوە بۆ ١٥ % کەمکرایەوە.  بانکی ناوەندیی تورکیا رۆژی ١٦ی کانوونی یەکەمی ٢٠٢١ سوودی بانکی لە ١٥ % ـەوە بۆ ١٤ % کەمکردبووەوە، لە کۆبوونەوەی مانگی کانوونی دووەمی ٢٠٢٢ ـەوە تاوەکو ئێستا بۆ پێنج جار لەسەر یەک سوودی بانکی لە رێژەی ١٤ % دا هێشتووەتەوە.
