لێکۆڵینەوەکە لە شێوەی راپرسی لە رێگەی تەلەفۆنەوە کراوە، لەو ١٠٦٧ کەسەی بۆچوونیان وەرگیراوە، ٦٦.١ % یان داوای گەڕاندنەوەی پەنابەرانی سوورییان کردووە. ٤٥.٥ % ی خەڵک لە تورکیا پێیانوایە سوورییەکانی نیشتەجێی تورکیان مەترسیدارن و لە داهاتوودا دەبنە هۆکاری دروستبوونی کێشە. ٤١.٧ % ی ئەو کەسانەی رای خۆیان دەربڕیوە وەک بارگرانی دەڕواننە پەنابەرانی سووری لە تورکیا.   بەگوێرەی دەنگدەری پارتە سیاسییەکان بێت، لێکۆڵینەوەکە دەریخستووە، ٤٢.٦ % ی دەنگدەری ئاکپارتی دەڵێن پێویستە پەنابەرانی سووری بگەڕێنەوە وڵاتی خۆیان، ٥٠ % ی دەنگدەرانی سەرجەم پارتەکانی دیکەی تورکیا پێیانوایە، پێویستە پەنابەرانی سووری بگەڕێنەوە وڵاتی خۆیان.   ٤٨.٣ % ی بەشداربووانی راپرسییەکە لە تورکیا لە پرسی پەنابەرانی سوورییدا دەڵێن سیاسەتی حکومەت "زۆر هەڵەیە"، ١٨.٥ % یش دەڵێن "هەڵەیە".   بەگوێرەی دەنگدەری پارتە سیاسییەکان بێت، ٤٦.٣ % ی دەنگدەری ئاکپارتی، ٨٨ % ی دەنگدەری جەهەپە، ٦٦.٦ % ی دەنگدەری مەهەپە، ٨٨.١ % دەنگدەری ئیی پارتی و ٩١.٦ % ی دەنگدەری هەدەپە پێیانوایە سیاسەتی حکومەت لە پرسی پەنابەرانی سووری دا "هەڵەیە".   لە ئەگەری کۆتاییهاتنی شەڕ لە سووریا ٨٤.٢ % ی بەشداربووانی لێکۆڵینەوەکە دەڵێن "با سوورییەکان بگەڕێنەوە وڵاتی خۆیان".   لە وەڵامی پرسیاری "چۆن پێناسەی سوورییەکان دەکەن؟ " بەشی زۆری وەڵامەکان نەرێنین. زیاتر لە ٧٠ % ی خەڵک پێیانوایە "سوورییەکان پاک و خاوێن نین، جێگەی متمانە نین و کەسایەتیی بەرز نین". ٥٧.٥ % پێیانوایە "سوورییەکان ئیشکەر نین". ٥٥.٧ % ی بەشداربووانی لێکۆڵینەوەکە "نایانەوێت ببنە دراوسێی کەسێکی سووری"، هەروەها ٧١.٥ % ی دەڵێن "کاتێک ناچار دەبین" پەیوەندی لەگەڵ سوورییەکان دادەنێین.   لە وەڵامی پرسیاری "کێشەت لەگەڵ سوورییەکاندا هەیە؟ " ٤٩.٧ % بە "نەخێر" وەڵامیان داوەتەوە. ٣٥.١ % دەڵێن "هەندێکجار" کێشەم لەگەڵ سوورییەکاندا هەیە، و ١٥.٣ % دەڵێن "کێشەم" لەگەڵ سوورییەکاندا هەیە.   ٧٧.٣ % ی ئەو کەسانەی لە لێکۆڵینەوەکەدا بەشداربوونە لە وەڵامی پرسیاری "لە پێنج ساڵی رابردوودا بەشێوەیەکی راستەوخۆ لەلایەن سوورییەکانەوە زیانت پێ گەیشتووە؟ " بە "نەخێر زیانم پێ نەگەیشتووە" وەڵامیان داوەتەوە. لە وەڵامی پرسیاری "هیچ کەسێک لە ئەندامانی خێزانەکەت زیانی پێگەیشتووە؟ "، ٧٨.١ % بە "نەخێر" وەڵامیان داوەتەوە. لە وەڵامی پرسیاری "کەسانی دەورووبەرت زیانی پێگەیشتووە؟ " ٤٦.٧ % بە "بەڵی" وەڵامیان داوەتەوە، لە بەرامبەردا ٤١.١ % بە "نەخێر" وەڵامیان داوەتەوە.
